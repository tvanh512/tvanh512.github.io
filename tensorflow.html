Tensorflow basic tutorial

Tensor Flow

Tensorflow is an useful tool to build neural network by Google. You can check the installation guide here. I use Mac OS, so I just simply go to Terminal and type "sudo pip install tensorflow".
Below code is suitable with Python 2.7 and Tensorflow 0.12. Newer versions of Python and Tensorflow might need some small modifications.
Firstly, let's start with a Hello World program:

In [3]:
import tensorflow as tf
# Create session with the TensorFlow run time
with tf.Session() as sess:
    # Define some ops
    c = tf.constant("Hello")
    print sess.run(c)
sess.close()

Hello
